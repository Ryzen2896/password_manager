# Password Manager

A simple CLI password manager written in Rust using AES-GCM encryption.

## Features

- Secure password encryption/decryption with a master password
- Password generation with customizable length
- Local storage of encrypted passwords in JSON file

## Requirements

- Rust 1.65+ (or latest stable)
- Dependencies managed via Cargo
